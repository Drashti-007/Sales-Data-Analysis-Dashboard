# Sales Data Analysis Dashboard

An end-to-end Data Analytics project that analyzes Superstore sales data using **Python (Pandas & Matplotlib)** and **Microsoft Power BI**. The project focuses on data cleaning, exploratory data analysis (EDA), visualization, and building an interactive dashboard to uncover business insights.

## Project Overview

The objective of this project is to analyze retail sales data to answer key business questions such as:

- Which region generates the highest sales?
- Which product categories are the most profitable?
- How have sales changed over time?
- Which products contribute the most profit?

The final output is an interactive Power BI dashboard that allows users to explore sales performance using filters.

## Data Cleaning

The dataset was cleaned using Pandas by:

- Removing duplicate records
- Handling missing values
- Converting date columns to DateTime format
- Creating Year and Month columns
- Preparing data for visualization

## Exploratory Data Analysis

The following analyses were performed:

- Total Sales
- Total Profit
- Average Sales
- Total Orders
- Sales by Region
- Sales by Category
- Profit by Category
- Monthly Sales Trend
- Yearly Sales Trend
- Top 10 Products by Profit

Python libraries such as **Pandas** and **Matplotlib** were used for data analysis and visualization.

## Power BI Dashboard

The interactive dashboard includes:

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Average Sales KPI
- Sales Trend Over Time
- Sales by Region
- Sales by Category
- Top 10 Products by Profit
- Interactive slicers for Region, Category, and Year

## Key Insights

- The **West** region generated the highest sales.
- **Technology** was the highest-selling product category.
- Technology also produced the highest overall profit.
- Sales showed an overall upward trend between **2014 and 2017**.
- A small number of products contributed significantly to total profit.
